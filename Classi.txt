Classi per il gioco:
- Card 
	int value (da uno a dieci) 
	Character seed ( B -> bastoni, C -> coppe, D -> denari, S -> spade)

- Deck
	Arraylist<Card> cards (una lista contenente le carte del mazzo)

- Table
	Arraylist<Card> cards (una lista contenente le carte sul tavolo)
	Player[] players (un array contente i giocatori della partita cosi da avere un ordine per i ruoli)

- Hand
	Arraylist<Card> cards (una lista contenente le carte in mano al giocatore)
	
- DeckPlayer
	Arraylist<Card> cards (una lista contenente il mazzetto vinto dal giocatore)
	Player player 
	
- Player 
	String Id (nome univoco per ogni account)
